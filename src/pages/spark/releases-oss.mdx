## Compatibility with Apache Spark

The following table lists Delta Lake versions and their compatible Apache Spark versions.

| Delta Lake version | Apache Spark version |
| ------------------ | -------------------- |
| 1.2.x              | 3.2.x                |
| 1.1.x              | 3.2.x                |
| 1.0.x              | 3.1.x                |
| 0.7.x and 0.8.x    | 3.0.x                |
| Below 0.7.0        | 2.4.2 - 2.4._latest_ |
